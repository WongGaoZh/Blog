#### 查找文件
推荐使用locate

```
       which  查看可执行文件的位置。
       whereis 查看文件的位置。 
       locate   配合数据库查看文件位置。
       find   实际搜寻硬盘查询文件名称。
which命令的作用是，在PATH变量指定的路径中，搜索某个系统命令的位置，并且返回第一个搜索结果。
也就是说，使用which命令，就可以看到某个系统命令是否存在，以及执行的到底是哪一个位置的命令。 


where is  查看bin下面有那些命令


TIPS：在用搜索指令时，请记得用sudo且cd到  /  目录下，这样才不会找漏文件，这个很重要！！！！
```
